A-CLA: Adaptive Cellular Learning Automata for Community Detection

This repository contains the reference implementation of A-CLA (Adaptive Cellular Learning Automata), a learning-automata–based framework for community detection in complex networks.

The code is provided to support reproducibility and transparency of the experimental results reported in the manuscript:

“A-CLA: A Novel Adaptive Cellular Learning Automata Approach for Uncovering Communities in Complex Networks”
(submitted to Computer Networks).

Requirements:

Python ≥ 3.8

NumPy

NetworkX

SciPy

scikit-learn

Matplotlib

You can install dependencies using:

pip install -r requirements.txt
